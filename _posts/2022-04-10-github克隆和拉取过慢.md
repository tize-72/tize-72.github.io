### 1.修改host文件

[GitHub: Where the world builds software · GitHub (ipaddress.com)](https://ipaddress.com/website/github.com)

分别搜索

 github.global.[ssl](https://so.csdn.net/so/search?q=ssl&spm=1001.2101.3001.7020).fastly.net 

 github.com

然后将对应的ip地址放在hts文件中

Windows对应的文件地址：**C:\Windows\System32\drivers\etc**

刷新就可以

### 2.取消代理

```
git config --global --unset http.proxy

git config --global --unset https.proxy
```

取消代理之后，再进行push就可以成功$ git push -u origin master -f



设置代理：

```
git config --global http.proxy http://127.0.0.1:1080
 
git config --global https.proxy http://127.0.0.1:1080
```
