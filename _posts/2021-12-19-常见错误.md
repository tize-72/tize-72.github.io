# 错误总结

- #### AssertionError: Torch not compiled with CUDA enabled

- [x] 由于下载的时候就没有下载Gpu加速版本的torch导致的，试一下重装torch

  进了torch的官网https://pytorch.org/get-started/locally/

  试了试这个命令，但是使用内核运行的时候还是错误

  ```
  conda install pytorch torchvision torchaudio cudatoolkit=11.3 -c pytorch
  ```

- [x] 看了下自己电脑的英伟达，说的是cuda版本是11.4，准备试试安装一个11.4的

  不行，失败了，因为安装的过程很慢

- [x] 试一下另一个命令，看看这个方案能不能行

  ```
  pip3 install torch==1.10.1+cu113 torchvision==0.11.2+cu113 torchaudio===0.10.1+cu113 -f https://download.pytorch.org/whl/cu113/torch_stable.html
  ```

  不行，安装失败

- #### Original error was: DLL load failed: 找不到指定的模块。

这个错误是python在终端运行的时候出错

- [x] 以下是打印路径的

  ```python
  import sys
  for i in range(len(sys.path)):
      print(sys.path[i])
  ```

- #### Importing the numpy C-extensions failed. This error can happen for many reasons, often due to issues with your setup or how NumPy was installed.

这个错误是导入numpy失败，然后一篇博客说是因为numpy的版本与python不一致导致的[这个](https://blog.csdn.net/qq_41032721/article/details/120229799)，准备试试

- [x] 修改numpy版本为1.18.1 之后成功了

- #### RuntimeError: Expected all tensors to be on the same device, but found at least two devices, cuda:1 and cpu!

意思是说没有将所有的tensor都放在同一个device上，程序中有的地方跑的cpu，有的地方跑的gpu

- [ ] 参考了一个python学习笔记，原来之前直接把.to(device)写在一个公式的后边是不对的，因为那样没有区分开来变量，

  [python学习笔记]: https://blog.csdn.net/weixin_41503889/article/details/120349496?spm=1001.2101.3001.6650.1&amp;utm_medium=distribute.pc_relevant.none-task-blog-2%7Edefault%7ECTRLIST%7Edefault-1.fixedcolumn&amp;depth_1-utm_source=distribute.pc_relevant.none-task-blog-2%7Edefault%7ECTRLIST%7Edefault-1.fixedcolumn

  改了之后，程序可以跑了，
