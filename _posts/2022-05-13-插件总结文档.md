### Conda包管理

#### pipreqs

能够将环境中的所有包都导出到一个requirement文件中去

1.安装pipreqs

```
pip install pipreqs
```

2.命令生成

（1）先 cd 到程序所在的文件夹位置
（2）执行下面的语句，Windows下得加encoding=utf8，不然会出现编码问题。

```
pipreqs ./ --encoding=utf8 
```

然后生成之后，就可以使用pip install -r requirements.txt 进行所有包的安装
