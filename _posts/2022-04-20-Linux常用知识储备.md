### Linux压缩

tar 常用参数

```
    -c :打包压缩
    -x :解压
    -z :压缩格式为.gz
    -v :显示过程
    -f :指定压缩文件名
```

压缩

```
tar -cvf 压缩包名.tar 源文件
```

压缩为.tar.gz文件

```
tar -zcvf 压缩包名.tar.gz 源文件
```

解压.tar.gz文件

```
tar -zxvf 压缩包名.tar.gz
```
