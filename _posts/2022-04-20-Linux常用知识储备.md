### Linux压缩

tar 常用参数

```
    -c :打包压缩
    -x :解压
    -z :压缩格式为.gz
    -v :显示过程
    -f :指定压缩文件名
```

压缩

```
tar -cvf 压缩包名.tar 源文件
```

压缩为.tar.gz文件

```
tar -zcvf 压缩包名.tar.gz 源文件
```

解压.tar.gz文件

```
tar -zxvf 压缩包名.tar.gz
```

### 查看文件相关

当磁盘大小超过标准时会有报警提示，这时如果掌握df和du命令是非常明智的选择。

df可以查看一级文件夹大小、使用比例、档案系统及其挂入点，但对文件却无能为力。

du可以查看文件及文件夹的大小。

df命令可以显示目前所有文件系统的可用空间及使用情形

![2026201-20211111154345800-1817809053.png](https://s2.loli.net/2022/05/11/LqiuIhvonXMVYTj.png)

命令du -h –max-depth=1 *，可以查看当前目录下各文件、文件夹的大小，这个比较实用。

![2026201-20211111154915166-1158050602.png](https://s2.loli.net/2022/05/11/PC1aE5flDNnB8IW.png)

查询当前目录总大小可以使用du -sh，其中s代表统计汇总的意思，即只输出一个总和大小。

![2026201-20211111155331114-1465540716.png](https://s2.loli.net/2022/05/11/XjxBfrDZnzNO13m.png)

对于指定文件夹也可以指定显示层次深度，如du -h --max-depth=0 PLSQL/及du -h --max-depth=1 PLQSL/

![2026201-20211111155743925-317958951.png](https://s2.loli.net/2022/05/11/GxNOHWe7cd3TAM6.png)

### scp文件复制

远程服务器传回本地

在本机上执行scp，把远端的服务器文件拷贝到本机上：

scp root@192.168.1.147:/super/dns.pcap /

本机文件拷贝到远程服务器

把本机的文件传给目的服务器：

scp get66.pcap root@192.168.1.147:/super
