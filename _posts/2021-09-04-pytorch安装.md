### 二、pytorch安装前准备

#### 1.使用清华镜像源网站

切换到国内的镜像源（切换是因为使用国内镜像下载的时候网速比较快，若使用外网服务器下载太慢了，

若已经科学上网，可以无视该step）
windows10下使用Win+R键打开命令行，分别输入下列命令，每输入一行之后就回车一下

```
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/

conda config --set show_channel_urls yes

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/
```

#### 2.创建并切换Pytorch环境

创建pytorch的环境，使用的是python3.7，依旧在命令行中输入下列语句后回车：

```
conda create -n pytorch python=3.7
```

然后要检查这个环境是否已经安装好了**并且切换到这个环境**

输入下列语句后回车：

```
conda info --envs
```

出现下图时，可以看到有一个pytorch的conda环境

![](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221910875.png)

下面要切换到这个环境

>  [参考网址](https://blog.csdn.net/weixin_41297324/article/details/109478948?ops_request_misc=%257B%2522request%255Fid%2522%253A%2522162954777116780366539920%2522%252C%2522scm%2522%253A%252220140713.130102334.pc%255Fall.%2522%257D&request_id=162954777116780366539920&biz_id=0&utm_medium=distribute.pc_search_result.none-task-blog-2~all~first_rank_ecpm_v1~rank_v29_ecpm-2-109478948.pc_search_download_positive&utm_term=conda+%E8%BF%9B%E5%85%A5%E5%88%9B%E5%BB%BA%E7%9A%84pytorch%E7%8E%AF%E5%A2%83&spm=1018.2226.3001.4187)

继续在命令行中输入下列语句后回车

```
conda activate pytorch
```

出现下图：

![](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221910788.png)

可以看到已经切换到pytorch环境下了

### 三、安装pytorch

#### 1.首先确定自己的CUDA版本

> [CUDA匹配参考网址](https://blog.csdn.net/qq_42947110/article/details/108832732?ops_request_misc=%257B%2522request%255Fid%2522%253A%2522162954287216780366526960%2522%252C%2522scm%2522%253A%252220140713.130102334..%2522%257D&request_id=162954287216780366526960&biz_id=0&utm_medium=distribute.pc_search_result.none-task-blog-2~all~sobaiduend~default-1-108832732.pc_search_download_positive&utm_term=pytorch%E7%89%88%E6%9C%AC%E9%80%89%E6%8B%A9&spm=1018.2226.3001.4187)

打开NVIDIA管理面板（一般都在任务栏的右边），点击帮助

![在这里插入图片描述](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221910207.png)

进入“系统信息”，显卡及驱动版本如红框所示，驱动版本为398.26

![](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221910546.png)

然后对照着CUDA匹配表，网址为[CUDA匹配](https://docs.nvidia.com/cuda/cuda-toolkit-release-notes/index.html#cuda-major-component-versions)

![在这里插入图片描述](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221910012.png)

可以看到表中对应的CUDA版本是**9.2**

#### 2.安装pytorch

在上述这张图的命令行状态下（**注意看前缀一定要是pytorch环境**）

![image-20210822104759348](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221910968.png)

输入下列命令并回车，若出来需要输入y或者n的时候，直接输入y并回车就好

```
conda install pytorch torchvision cudatoolkit=9.2 -c pytorch
```

等待执行完毕，pytorch就安装完成了

#### 3.检测pytorch是否安装成功

命令行中（pytorch环境下），输入python，进入python代码的编辑模式

![](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221910771.png)

然后分别输入下列python代码

```python
import torch
import torchvision
print(torch.__version__)
```

出现下图

![](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221910755.png)

可以看到打印出了torch的版本号1.5.1，证明安装pytorch成功

### 四、让Jupyter Notebook支持pytorch

此时单击Jupyter Notebook或者在命令行中输入Jupyter Notebook都会打开Jupyter Notebook，而且自动搭载好了python3的环境，普通的python代码已经可以运行了。下图中点击python3

![在这里插入图片描述](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221910934.png)

输入代码后点击运行：

![](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221910940.png)

可以看到一般的python代码可以运行了，但是此时还没有pytorch库，所以输入这个代码会报错

![](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221910918.png)

首先，以管理员身份打开Anaconda Prompt，然后安装插件。
命令行输入

```
conda install nb_conda
```

遇到提示的话，输入y继续回车安装。

然后进入创建的pytorch环境（**这一步忘了的可以看第二章的2，看如何切换到pytorch环境**），命令行输入

```
conda install ipykernel
```

后回车，等待安装成功

![在这里插入图片描述](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221911957.png)

安装成功后，再次打开Jupyter，查看语言的地方

![](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221911414.png)

可以看到，标蓝色框的是多出来了两个语言，点击进入第二个有pytorch的

继续输入之前在命令行中输入过的python代码进行验证，发现也可以输出torch的版本号

![](https://gitee.com/Tize-72/picgo-imgs/raw/master/img//202108221911910.png)
